# Ecommerce API application

This is a Ecommerce API application ready out of the box (after instaliation).

## Clone Repo

    git clone https://github.com/garbox/RESTful-API-Ecom

## Composer install

    composer install

## Run migrations

    php artisan migrate

## Test Application

    php artisan test

## Get First API Key

    php artisan app:first-api-token
